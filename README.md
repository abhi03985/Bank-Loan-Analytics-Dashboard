# 📌 Project Overview

This project demonstrates how **Microsoft Excel** can be used to build a professional **Finance MIS Dashboard** for analyzing bank loan performance.

The dashboard helps management monitor loan applications, funded amounts, repayments, borrower risk, and lending performance through interactive visualizations and KPI reporting.

---

# 🎯 Business Objective

- Monitor overall loan portfolio performance
- Track funded amount and repayments
- Identify Good Loans and Bad Loans
- Analyze loan trends over time
- Evaluate borrower profiles
- Support data-driven lending decisions

---

# 📊 Dashboard Features

### Executive KPIs

- 📄 Total Loan Applications
- 💰 Total Funded Amount
- 💵 Total Amount Received
- 📈 Average Interest Rate
- 📉 Average Debt-to-Income (DTI)

### Analysis

- ✅ Good Loan vs Bad Loan Analysis
- 📅 Monthly Loan Trends
- 🏦 Loan Status Analysis
- 🏠 Home Ownership Analysis
- ⭐ Grade-wise Performance
- 🎯 Loan Purpose Analysis
- 🌍 State-wise Loan Distribution
