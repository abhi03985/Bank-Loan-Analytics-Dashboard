**# 💰 Finance MIS Dashboard | Bank Loan Analytics

An interactive **Finance MIS Dashboard** built in **Microsoft Excel** to analyze and monitor loan portfolio performance. The dashboard provides a comprehensive overview of loan applications, funded amounts, repayments, interest rates, and borrower risk metrics through dynamic KPIs, Pivot Tables, and interactive visualizations.

---

## 📌 Project Objective

The objective of this project is to create an interactive MIS dashboard that enables management to:

- Monitor overall loan portfolio performance
- Track loan applications and funded amounts
- Analyze repayment performance
- Compare Good Loans vs Bad Loans
- Monitor Interest Rate and Debt-to-Income (DTI)
- Generate business insights for better lending decisions
